# Invitación Digital de Boda — Eunice & Magdiel

Sistema web para administrar una invitación digital de boda con confirmación de asistencia, gestión de invitados, QR personalizado, check-in, mesas, reportes y panel administrativo.

## Funciones principales

- Invitación digital personalizada.
- Confirmación de asistencia RSVP.
- Invitaciones por token único.
- Panel administrativo.
- Gestión de invitados.
- Generación de QR.
- Check-in con cámara.
- Gestión de mesas.
- Plano visual del salón.
- Reportes imprimibles.
- Exportación a Excel.
- Administración de multimedia.
- Configuración de la boda desde el panel.
